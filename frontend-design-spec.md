# 个人综合应用系统 - 前端设计方案

## 设计理念与美学方向

### 核心定位

本系统定位为**「数字生活收藏馆」**——一个优雅的个人数据管理中心，将音声、小说、偶像收藏、手工素材、云端文件五大私人领域整合为统一的视觉体验。设计语言借鉴 Apple 生态的克制美学，结合莫兰迪色系的柔和气质，营造「安静而精致」的私人空间感。

### 视觉关键词

| 关键词 | 诠释 |
|--------|------|
| 轻盈 | 大量留白，元素间呼吸感强，避免拥挤 |
| 柔和 | 圆角过渡、渐变阴影、绒雾质感 |
| 层次 | 微妙的卡片叠层、玻璃拟态、分区清晰 |
| 沉浸 | 沉浸式内容消费体验，功能服务于内容 |
| 温暖 | 莫兰迪色调传递亲切感，非冷峻的工具感 |

### 差异化记忆点

**「毛玻璃容器」**——所有功能卡片采用 frosted glass 效果（backdrop-filter: blur），营造漂浮于浅色背景之上的轻盈感。用户滑过卡片时，卡片轻微上浮并发光，如同桌面上的便签被手指触碰。

## 四、组件设计规范

### 4.1 毛玻璃卡片 (GlassCard)

**视觉特征**
- 背景: `rgba(255, 255, 255, 0.72)`
- 模糊: `backdrop-filter: blur(20px) saturate(180%)`
- 圆角: 20px
- 阴影: `0 8px 32px rgba(0, 0, 0, 0.08)`
- 边框: `1px solid rgba(255, 255, 255, 0.4)`

**悬停状态**
```css
transform: translateY(-4px);
box-shadow: 0 16px 48px rgba(0, 0, 0, 0.12);
transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1);
```

### 4.2 按钮系统

#### 主按钮 (Primary)
```

禁用:   背景 #C4C0BE | 文字 #8B8685 | 不可点击
```

## 九、特殊场景设计

### 9.1 深色模式

```
触发方式: 
- 系统跟随 (prefers-color-scheme)
- 手动切换 (Header 或设置页)

切换动画:
- 颜色过渡 0.3s ease
- 避免纯黑白闪烁

特殊处理:
- 图片: 使用 CSS filter 或多版本
- 阴影: 调整透明度
- 毛玻璃: 背景色加深
```

### 9.2 无障碍设计

```
颜色对比度: AA 级 (4.5:1)

点击区域: 最小 44x44px

减少动画: 支持 prefers-reduced-motion
字体缩放: 支持 200% 缩放
```
